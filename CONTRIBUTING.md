# Contributing

Contributions are welcome — especially new quotes!

## Adding Quotes

The most valuable contribution is a well-sourced literary quote:

1. The exact time must appear in the text
2. Include: quote, author, book title, language
3. Prefer published fiction, poetry, or essays
4. Verify the quote against the original source

Submit quotes as JSON entries or open an issue.

## Code Contributions

1. Fork the repository
2. Create a feature branch
3. Commit your changes
4. Open a Pull Request

## Design Principles

- Typography is sacred — don't clutter the screen
- Performance matters — this runs on nightstand tablets
- Accessibility: readable contrast, screen reader support
